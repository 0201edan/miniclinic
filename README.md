# 輔大醫資期末專案：MiniClinic 社區診所掛號系統

這個專案是我大一這學期在 Java 程式設計課寫的期末整合作業。主要做的是一個診所的掛號系統，除了可以幫病人掛號、讓醫生登入看後台之外，也把前面幾週學到的資料庫持久化跟登入安全機制全部串起來。

## 專案裡有做出來的功能
- **掛號功能：** 可以幫來看病的人預約掛號，也可以查到他們的掛號紀錄。
- **醫生和病人的資料管理：** 負責把診所裡的醫生跟病人的基本資料管好（做 CRUD 的操作）。
- **後台 Dashboard 統計（這週新加的！）：** 醫生登入進去之後，可以看到今天掛號的總人數，還有分別有多少人是正在等看診（BOOKED）、看完診（COMPLETED）或取消（CANCELLED）。
- **登入安全防護：** 病人的個資很重要，所以醫生的密碼有用 BCrypt 偷偷加密過，而且有用 Interceptor 攔截器做保護，如果沒有登入就想偷看後台，會直接被踢回登入頁面。

## 這次用到的技術跟工具
- **寫程式寫後端：** 用 Java 的 Spring Boot 框架來寫。
- **存資料的地方（資料庫）：** 本機開發是用比較輕量的 SQLite，不過為了部署到 Render 雲端，正式上線版有換成 PostgreSQL 資料庫。
- **資料操作：** 用 Spring Data JPA 來跟資料庫對話，不用自己苦哈哈寫一堆複雜的 SQL 指令。
- **網頁畫面：** 畫面是用前端 Thymeleaf 模板套入 HTML 來做動態渲染。
- **雲端部署：** 專案有寫 Dockerfile 弄成容器化，最後成功丟到 Render 雲端平台上跑。

1. 先把這個專案 clone 到你本機的電腦。
2. 用 VS Code 或任何 IDE 打開最內層有 pom.xml 的資料夾。
3. 打開終端機輸入下面這行指令，就可以在本機跑起來了：

./mvnw spring-boot:run
